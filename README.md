# Energieverbrauchsprognose mit Transformer

Dieses Repository enthält:
- Python-Module für Datenaufbereitung, Modellierung und Auswertung
- Ein Jupyter Notebook zur Präsentation der Ergebnisse

## Struktur

- `energy_forecast_transformer/` – Python-Paket mit Kernfunktionen
- `notebooks/` – Präsentationsnotebooks
- `data/` – Roh- und Zwischendaten
- `README.md` – Projektbeschreibung

## Setup

1. Python-Umgebung erstellen (z.B. mit venv oder conda)
2. Abhängigkeiten installieren:
   ```
   pip install -r requirements.txt
   ```
3. Notebook ausführen

## Beispiel: Import im Notebook
```python
from energy_forecast_transformer.preprocessing import load_and_clean_data
```
